# Reliance Jio Customer Insights Dashboard

## Project Overview

This project is a Power BI dashboard created to analyse telecom customer data and generate useful insights related to customer behaviour, revenue and churn trends.

The dashboard is designed using a Jio-inspired theme and presents data in a clear and interactive format.

## Objectives

- Analyse total customers and monthly revenue
- Identify churned customers
- Compare users across regions
- Study contract plan preferences
- Analyse payment methods and internet services

## Dashboard Features

### KPI Cards
- Total Customers
- Monthly Revenue
- Average Revenue per User
- Churned Users

### Visuals
- Customers by Contract Type
- Users by Region
- Churn by Payment Method
- Internet Service Split
- Tenure vs Monthly Charges
- Customer Details Table

### Filters / Slicers
- Region
- Contract
- Churn
